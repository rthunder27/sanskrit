# sanskrit
The purpose of this repo is to build an app to help one self-learn Sanskrit.

Features
1)Devanagari flash cards, learn how to identify and pronounce each letter. One side is just the character, the other has the english transliteration, phonetic pronouncation, and an example of it in a word.
2) Use the touch screen to practice drawing each charachter.
3) Learn pronounciation (maybe)

Perhaps vocab and grammar after exploring these features, definetely 1, probably 2, and maybe 3.
